## 🎯 Features

- **Real-time Video Calls** — Peer-to-peer video communication using WebRTC
- **Screen Sharing** — Share your screen with other participants
- **Multiple Participants** — Support for group video meetings
- **Firebase Integration** — Real-time database for signaling and room management
- **Responsive Design** — Works across desktop and mobile browsers

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- Firebase account (for configuration)
- Modern browser with WebRTC support

### Installation

1. Clone the repository
```bash
git clone https://github.com/Raj-Kori/Video-Conferencing-Website.git
cd Video-Conferencing-Website/webrtc-firebase-demo-main
```

2. Install dependencies
```bash
npm install
```

3. Configure Firebase
   - Create a Firebase project
   - Add your Firebase config to the application

4. Start the server
```bash
npm start
```

5. Open `http://localhost:3000` in your browser
